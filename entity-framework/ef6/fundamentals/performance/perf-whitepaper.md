-   <span data-ttu-id="551cb-822">Tabelle pro Hierarchie (TPH) – wobei jeder Vererbungs Satz einer Tabelle mit einer diskriminatorspalte zugeordnet wird, um anzugeben, welcher bestimmte Typ in der Hierarchie in der Zeile dargestellt wird.</span><span class="sxs-lookup"><span data-stu-id="551cb-822">Table per Hierarchy (TPH) – where each inheritance set maps to a table with a discriminator column to indicate which particular type in the hierarchy is being represented in the row.</span></span>
